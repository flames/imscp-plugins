i-mscp-plugins
=========================

Set of plugins for i-MSCP. These plugins files are only compatible with i-MSCP >= **1.2.0**.

## Listener files

Below, you can find a list of all plugins which are available in that repository, and their respective purpose.

### RawPasswd

The **RawPAsswd** plugin saves the clear passwords of panel-users into a separate column, when users password is changed by admin, reseller or user him self.

To install the plugin, upload it to **/var/www/imscp/gui/plugins** directory, and install/enable through i-MSCP plugin management interface
